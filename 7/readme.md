# Android Lazy Multi-Counter App

This app demonstrates Jetpack Compose state management and dynamic UI creation using `LazyColumn`. The app displays a scrollable list of counters, each of which can be independently incremented or decremented.

## Features

- Scrollable list of counters using `LazyColumn`
- Independent counter values maintained by Compose state
- Add or remove counters dynamically at runtime
- Simple Material 3 design

## Technologies

- Kotlin
- Jetpack Compose
- Material 3
