# Kotlin Functions and Higher-Order Functions

## Description

This simple Kotlin program demonstrates how to use:

- `filter { }` to select even numbers
- `map { }` to calculate the square of each number
- `reduce { acc, n -> ... }` to compute the sum of all numbers

The program works on a list of integers from 1 to 100 and prints:

- All even numbers
- The square of each number
- The total sum of all numbers

## Files

- `Main.kt` — main Kotlin source file
- `output.txt` — program output
