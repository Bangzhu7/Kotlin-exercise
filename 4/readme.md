# Android Product Banner App

This is a simple Android application that displays a product banner. It was created as a guided project to learn the basics of Android development with Jetpack Compose.

## Features

- Displays a background product image.
- Overlays text for the product name, company name, and contact information.
- Uses a `Box` layout for layering and a `Column` for vertical text alignment.
