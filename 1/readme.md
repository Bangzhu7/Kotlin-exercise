# Prime and Even Number List

This Kotlin console program:

- Finds the first 50 prime numbers and stores them in a list.
- Prints the list.
- Creates a list of the first 50 even numbers.
- Adds them to the prime list and prints the combined list.

## Files

- `Main.kt` — Kotlin source code
- `output.txt` — Program output

## How to Run

1. Open project in Android Studio.
2. Click **Run ▶️** to execute the program.
